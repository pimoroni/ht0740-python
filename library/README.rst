Installing
==========

Stable library from PyPi:

-  Just run ``sudo pip install ht0740``

Latest/development library from GitHub:

-  ``git clone https://github.com/pimoroni/ht0740-python``
-  ``cd ht0740-python``
-  ``sudo ./install.sh``
